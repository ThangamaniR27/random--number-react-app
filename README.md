# 🎲 Random Number Generator (React)

A simple React application that generates a random number when you click a button.

---

## 🚀 Features

* Generate random number (1–100)
* Placeholder message before generating
* Clean and simple UI
* Built using React Hooks (`useState`)

---

## 🛠️ Tech Stack

* React JS
* JavaScript (ES6)
* HTML5
* CSS (Inline Styling)

---

## 📂 Folder Structure

```bash
src/
 ├── App.js
 └── components/
      └── RandomNumber.js
```

---

## ⚙️ Setup Instructions

1. Clone the repository

```bash
git clone https://github.com/ThangamaniR27/random--number-react-app.git
```

2. Go to project folder

```bash
cd random-number
```

3. Install dependencies

```bash
npm install
```

4. Start the app

```bash
npm start
```

---

## 🎯 How It Works

* Click the **Generate** button
* A random number between 1 and 100 will be displayed
* Uses `Math.random()` and React state
